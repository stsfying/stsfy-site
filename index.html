Satisfying software tools for accountants

stsfy is a curated selection of community-developed software tools built for accountants. Every tool here is open core - free and open to inspect, adapt, and extend.

  • Build together, don't reinvent alone. Accountants solving the same problems independently is a real waste of effort. By sharing tools and collaborating, we can be more efficient and create higher-quality tools.
  • For growing businesses. These tools are aimed at start-ups, scale-ups and SMEs that need something simpler than the enterprise-grade platforms multinationals have the budget for.
  • Built LLM-first. Rather than being designed around a human clicking through a GUI, every tool here is built so an LLM can call it directly - and built to be LLM-agnostic, working with any frontier model rather than being locked to one provider.

Tools

HappyStats

In development

A local-first iXBRL statutory accounts tool. HappyStats imports a trial balance, maps it to the FRC IFRS 2024 taxonomy, and generates primary statements (SOFP, SOCI, SCF) plus disclosure notes - producing validated iXBRL and PDF output ready for UK Companies House filing.

All data stays on your own machine or shared drive. Nothing is sent to a server or the cloud.
